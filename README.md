# homebrew-isle

Official [Homebrew](https://brew.sh) tap for [Isle](https://github.com/TheVinizzz/Isle) — a Dynamic Island-style hub for the MacBook notch.

## Install

```bash
brew install --cask TheVinizzz/isle/isle
```

Or, in two steps if you'd like the tap available for `brew search`:

```bash
brew tap TheVinizzz/isle
brew install --cask isle
```

Homebrew clears the macOS quarantine flag automatically, so the install completes without any "Apple cannot verify…" prompts.

## Upgrade

```bash
brew upgrade --cask isle
```

## Uninstall

```bash
brew uninstall --cask isle
brew untap TheVinizzz/isle      # optional: remove the tap
```
